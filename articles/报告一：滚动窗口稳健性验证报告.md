## 三、Top 3 最佳窗口深度分析

```
权重几乎完全均匀分布，说明在这个窗口四个因子都在发挥正向作用。

```
在政策驱动行情中，alpha002（开盘-成交量相关性）和 alpha051（波动极差比）起到了关键作用。

### 🥉 W4 (2025-03-31 ~ 2025-06-30) — IR: 7.93

## 四、权重稳定性分析

### 因子权重在各窗口的分布

```
ILLIQ_20:  mean=0.173, std=0.051, range=[0.111, 0.245]
alpha002:  mean=0.341, std=0.067, range=[0.224, 0.405]  ← 最核心因子
MOM_60:    mean=0.178, std=0.052, range=[0.115, 0.274]
alpha051:  mean=0.308, std=0.075, range=[0.185, 0.387]
```

**关键发现**：
- **alpha002** 平均权重最高（34.1%），在所有窗口中都是主力因子
- **alpha051** 权重波动最大（std=0.075），说明其表现高度依赖市场状态
- **ILLIQ_20** 和 **MOM_60** 权重相对稳定，属于"常备军"
- 没有因子在所有窗口中都占据主导——**动态权重调整是必要的**

## 五、参数敏感性分析

### 不同配置对比

| 配置 | 均值超额 | IR均值 | 最大回撤 | 换手率 |
|------|:--------:|:------:|:--------:|:-----:|
| **Base (26w, Q=0.01, MW=0.6)** | 167.63% | 5.34 | -5.98% | 15.0x |
| Short (13w) | 167.63% | 5.34 | -5.98% | 15.0x |
| Long (39w) | 167.63% | 5.34 | -5.98% | 15.0x |
| **Kalman Q=0.005 (smoother)** 🏆 | **204.93%** | **6.55** | **-4.76%** | 15.0x |
| Kalman Q=0.02 (reactive) | 165.58% | 5.32 | -6.49% | 15.0x |
| **MaxW=0.4 (tighter)** | **189.10%** | **6.74** | -5.71% | 15.0x |
| MaxW=0.5 (medium) | 180.80% | 5.84 | -5.83% | 15.0x |

### 关键洞见

1. **Kalman Q=0.005（平滑版）** 是最优参数：平均超额比 Base 高出 **+37.30%**，最大回撤降低 **-1.22%**
2. **MaxW=0.4（更紧的权重约束）** 在风险调整后表现最佳（IR 6.74）
3. 窗口长度的调整（13w/26w/39w）对本策略影响不大，说明**因子信号本身具有跨周期的稳健性**
4. 预期管理：W1（2024 Q3）和 W3（2025 Q1）出现了负超额（-10.19%, -34.84%），集中在**市场风格切换期**

## 六、结论与展望

| 维度 | 结论 |

| **换手率** | ⚠️ 14.9x 超过常规 12x 上限，实盘需注意交易成本 |

### 后续优化方向

```
1️⃣ 采用 Kalman Q=0.005 平滑滤波替代默认 Q=0.01
2️⃣ 将 MaxW 从 0.6 下调至 0.4 以降低集中度风险
3️⃣ 在市场风格切换期（如 W1/W3 的表现）引入风格因子预警
4️⃣ 实盘部署时增加止损机制以控制尾部风险
```
