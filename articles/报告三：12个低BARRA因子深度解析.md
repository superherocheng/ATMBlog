## 一、设计哲学：为什么要低BARRA？

BARRA CNE5/CNE6 多因子模型是业界最广泛使用的风险模型，包含六大风格因子。当你的因子与BARRA因子高度相关时，超额收益会被归因于风险暴露而非Alpha。低BARRA相关因子的设计原则：

### 四大避让策略

```
1️⃣ 不使用基本面/财务数据
   → 避开 BARRA Value, Quality, Growth 因子

2️⃣ 不使用直接的价格动量
   → 使用高阶衍生量（MACD加速度、偏度）替代原始收益率

3️⃣ 不使用市值信息
   → 避开 BARRA Size 因子（所有因子基于百分比，不受市值影响）

4️⃣ 对技术指标进行非线性变换
   → 投票机制(theta函数)、分位数位置取代绝对值
   → 与 BARRA 的线性模型天然正交
```

### 预期优势

| 特性 | 说明 |
|------|------|
| **IC方向稳定性** | 与BARRA低相关意味着IC更稳定，不会随风格轮动大幅波动 |
| **组合多样性** | 与主流Barra因子互补，降低组合整体回撤 |
| **Alpha纯度** | 超额收益更难以被风险因子解释，真正意义上的Alpha |

## 三、12个因子逐一深度解析

### F01：RSRS修正斜率因子 — RSRS_Slope

**核心逻辑**：对过去18个交易日的最高价（High）和最低价（Low）进行 OLS 回归：

```
High = α + β × Low + ε
RSRS = β × R²
```

- `斜率 β = 1`：价格沿45度上行，多空力量均衡偏多
- `斜率 β > 1`：多头推动力强，高点弹性大于低点（看多）
- `斜率 β < 1`：空头压制，低点下移速度快于高点（看空）
- `R² 高`：支撑/阻力关系清晰，信号可靠
- `R² 低`：杂乱震荡，信号应过滤

**为什么低BARRA**：度量的是支撑/阻力弹性，而非价格方向本身。BARRA Momentum 看的是价格收益率（一阶），RSRS 看的是高低点回归结构（非动量维度）。

**因子类型**：微观结构/博弈因子  
**IC方向**：正向（斜率越高 → 支撑越强 → 未来收益越高）  
**预期IC均值**：0.02 ~ 0.04  
**ICIR**：0.3 ~ 0.6  
**有效窗口**：未来 1~5 个交易日

### F02：布林带宽度历史分位数因子 — BB_Width_Pct

**核心逻辑**：布林带宽度 = `(上轨 - 下轨) / 中轨`，取其在过去252日历史中的分位数：

```
BB_Width = 4 × σ / SMA(20)
BB_Pct   = Percentile(BB_Width, 252d)
```

- `分位数 = 90`：当前波动率处于历史高位 → 即将收窄 → 看空
- `分位数 = 10`：当前波动率处于历史低位 → 即将扩张 → 看多

**为什么低BARRA**：BARRA Volatility 度量的是收益率的绝对波动率（Sigma因子）。BB_Width_Pct 度量的是**波动率的相对位置**——它在自身历史中的分位数。这是两个完全不同的维度。一只股票可以波动率绝对值很低但处于其自身历史高位（BB宽度分位数高），此时 BARRA Sigma 看多低波动，但 BB_Width_Pct 看空波动扩张预期。

**因子类型**：波动率结构因子  
**IC方向**：反向（分位数越高 → 未来收益越低）  
**预期IC均值**：-0.01 ~ -0.03  
**ICIR**：0.2 ~ 0.5  
**有效窗口**：未来 5~20 个交易日

### F03：多指标技术综合评分因子 — Tech_Composite

**核心逻辑**：类似集成学习的**投票机制**——12个子信号各投 +1 / 0 / -1，汇总为综合得分：

```
趋势维度（8个信号）:
  ├─ SMA20: 价格在SMA之上 → +1, 之下 → -1
  ├─ EMA20: 价格在EMA之上 → +1, 之下 → -1
  ├─ WMA: 价格在WMA之上 → +1, 之下 → -1
  ├─ KAMA: 价格在KAMA之上 → +1, 之下 → -1
  ├─ MAMA: MAMA > FAMA → +1, 之下 → -1
  ├─ MACD柱: MACD柱在扩张 → +1, 收缩 → -1
  ├─ ADX: ADX > 25 → +1 (趋势确认), < 25 → -1
  └─ PPO: PPO加速 → +1, 减速 → -1

动量维度（3个信号）:
  ├─ RSI: > 70 → +1, < 30 → -1, 中间 → 0
  ├─ WILLR: > -20 → +1, < -80 → -1, 中间 → 0
  └─ MFI: > 80 → +1, < 20 → -1, 中间 → 0

波动率维度（1个信号）:
  └─ NATR: 低于均值 → +1 (低波看多), 高于均值 → -1
```

**为什么低BARRA**：BARRA 因子是线性模型（收益率、市值的线性组合）。**投票机制是阶梯函数**（非线性），12个子信号的二值/三值投票合成使得最终得分与任何线性因子的相关性都大幅降低。

**因子类型**：非线性技术复合因子  
**IC方向**：正向（得分越高 → 未来收益越强）  
**预期IC均值**：0.03 ~ 0.06  
**ICIR**：0.5 ~ 0.8  
**有效窗口**：未来 1~10 个交易日

### F04：量价背离因子 — Vol_Divergence

**核心逻辑**：`Divergence = Price_Change - Volume_Change`

```
价格变化率 = (Close_t - Close_{t-5}) / Close_{t-5}
量变化率   = (Volume_t - MA(Volume,5)) / MA(Volume,5)
背离因子   = 价格变化率 - 量变化率
```

- **价涨量缩（背离 > 0）**：上涨缺乏成交量支撑，虚涨，未来回调
- **价跌量增（背离 < 0）**：下跌伴随放量，恐慌盘出清，可能见底
- **量价同步**：趋势健康

**为什么低BARRA**：BARRA Liquidity 因子只看换手率的平方根——度量的是成交活跃度。不捕捉量价关系的**错位**。一只股票可以高换手率（BARRA Liquidity 高）但价涨量缩（背离因子看空），两者信号可以完全不同。

**因子类型**：技术博弈因子  
**IC方向**：反向（价涨量缩 → 负Alpha，价跌量增 → 正Alpha）  
**预期IC均值**：-0.02 ~ -0.04  
**ICIR**：0.4 ~ 0.6  
**有效窗口**：未来 1~5 个交易日

### F05：K线影线比率因子 — Shadow_Ratio

**核心逻辑**：计算近期K线的上影线和下影线比率：

```
上影线 = High - max(Open, Close)
下影线 = min(Open, Close) - Low
归一化比值 = mean(上影线/实体) / mean(下影线/实体)
最终值 = ln(比值 + 1)
```

- **上影线长**：多头曾大幅推高但被空头打回 → 上方阻力强 → 看空
- **下影线长**：空头曾大幅打压但被多头接回 → 下方支撑强 → 看多
- **比值 > 1**：卖方力量占优，反转风险高
- **比值 < 1**：买方力量占优，底部反转信号

**为什么低BARRA**：纯K线形态学，BARRA 模型完全不覆盖这种微观结构信息。上影线/下影线是从价格数据中挖掘出的**结构特征**，而非原始价格方向或波动率。

**因子类型**：微观结构因子  
**IC方向**：反向（上影线占比越高 → 未来收益越弱）  
**预期IC均值**：-0.01 ~ -0.02  
**ICIR**：0.2 ~ 0.4  
**有效窗口**：未来 1~3 个交易日

### F06：布林带位置因子 — BB_Position

**核心逻辑**：计算价格中枢在布林带(20,2)中的相对位置：

```
pivot = (H + L + C) / 3
位置 = (pivot - 下轨) / (上轨 - 下轨) × 100
```

- **位置 < 20（近下轨）**：价格处于近期低位，均值回复潜力大
- **位置 40~60（中轨附近）**：震荡中枢，方向不确定
- **位置 > 80（近上轨）**：价格处于近期高位，回调风险增加

**为什么低BARRA**：BARRA Momentum 看价格方向（12m-1m 收益率）。BB_Position 看价格在自身波动通道内的**相对位置**。一只下跌趋势的股票可以在下轨附近徘徊多日——动量因子持续看空，但 BB_Position 因子持续发出超卖信号。两者信号方向可以完全相反。

**因子类型**：均值回复因子  
**IC方向**：反向（位置越高 → 未来收益越低）  
**预期IC均值**：-0.02 ~ -0.04  
**ICIR**：0.3 ~ 0.6  
**有效窗口**：未来 3~10 个交易日

### F07：MACD加速度因子 — MACD_Accel

**核心逻辑**：动量的加速度——价格变化率的变化率的变化率：

```
step 1: EMA12 = EMA(Close, 12),  EMA26 = EMA(Close, 26)
step 2: MACD = EMA12 - EMA26
step 3: Signal = EMA(MACD, 9)
step 4: Histogram = MACD - Signal
step 5: Accel = Δ(Histogram_{t-1→t}) - Δ(Histogram_{t-2→t-1})
```

**层级关系**：
```
价格收益率                 BARRA Momentum        → 一阶
MACD柱(=快慢均线差)        准二阶                → 二阶
MACD柱变化率              一阶导                → 2.5阶
MACD加速度（本因子）       二阶导                → 三阶 🎯
```

**为什么低BARRA**：BARRA Momentum 是价格的 12m-1m 收益率（一阶）。MACD 柱 = 短期EMA - 长期EMA（准二阶）。MACD_Accel = MACD柱变化率的变化（三阶）——与原始价格收益率的相关性极低。一只股票可以上涨但 MACD 柱在收缩（动量减速），本因子看空，BARRA Momentum 看多。

**因子类型**：高阶动量因子  
**IC方向**：正向（MACD加速扩张 → 趋势强化 → 收益正向）  
**预期IC均值**：0.02 ~ 0.04  
**ICIR**：0.3 ~ 0.5  
**有效窗口**：未来 1~3 个交易日

### F08：标准化ATR偏离因子 — NATR_Deviation

**核心逻辑**：NATR（归一化ATR）相对自身252日均值的Z-Score偏离：

```
TR = max(High-Low, |High - Close_{t-1}|, |Low - Close_{t-1}|)
ATR(14) = EMA(TR, 14)
NATR = ATR(14) / Close × 100
Z    = (NATR(今) - mean(NATR, 252)) / std(NATR, 252)
```

- **Z > 0**：当前波动率高于自身历史均值 → 可能回归均值 → 看空
- **Z < 0**：当前波动率低于自身历史均值 → 可能回升 → 看多

**为什么低BARRA**：BARRA Volatility（Sigma）度量的是 **Close-to-Close 收益率的波动**。NATR 度量的是 **High-Low 日内幅度**。这是两个完全不同的波动度量维度。一只股票可以日间收益率稳定（BARRA Sigma 低）但日内振幅巨大（NATR 高），这与 A 股常见的"盘中剧烈震荡但收盘持平"现象高度吻合。

**因子类型**：波动率结构因子  
**IC方向**：反向（NATR偏离度越高 → 波动回归预期 → 反向信号）  
**预期IC均值**：-0.01 ~ -0.03  
**ICIR**：0.2 ~ 0.4  
**有效窗口**：未来 5~20 个交易日

### F09：收益率偏度因子 — Return_Skew

**核心逻辑**：计算过去20个交易日收益率序列的偏度：

```
Returns = (Close_t - Close_{t-1}) / Close_{t-1}
Skewness = moment(Returns, 3) / moment(Returns, 2)^(3/2)
```

- **正偏（右偏）**：偶有暴涨 → 彩票效应 → 投资者高估小概率事件 → 未来收益降低
- **负偏（左偏）**：偶有暴跌 → 恐惧效应 → 投资者过度规避 → 未来收益可能回升

**为什么低BARRA**：BARRA 没有任何一个因子对应收益率的高阶矩（偏度）。这是行为金融学中的经典异象——MAX效应（Bali et al., 2011）：过去出现极端正收益的股票（"彩票型"股票）未来跑输，因为投资者高估了小概率事件。

**因子类型**：行为金融因子  
**IC方向**：反向（正偏 → 彩票效应 → 未来收益低）  
**预期IC均值**：-0.02 ~ -0.04  
**ICIR**：0.3 ~ 0.5  
**有效窗口**：未来 1~20 个交易日

### F10：恐慌贪婪情绪因子 — Fear_Greed_Idx

**核心逻辑**：从4个维度综合打分：

```
D1: RSI动量维度:  |RSI(14) - 50|
    偏离50越远 → 情绪越极端（恐慌或贪婪）

D2: 成交量热度:  ln(Volume / MA(Volume, 20))
    放量 → 市场关注度高 → 情绪亢奋

D3: 波动率状态:  (ATR(14) - ATR_MA(60)) / ATR_MA(60)
    波动率扩张 → 分歧加大 → 情绪恐慌

D4: 价格位置:   (Close - LL(20)) / (HH(20) - LL(20))
    近上轨 → 贪婪, 近下轨 → 恐慌

Composite = D1 + D2 + D3 + D4（等权综合）
```

**为什么低BARRA**：行为金融学维度的情绪因子。BARRA 模型是纯粹的量化风险模型，没有任何行为金融变量。由于是四个维度的等权非线性合成，与任何单一 BARRA 因子的相关性都很低。

**因子类型**：情绪/行为金融因子  
**IC方向**：反向（贪婪 → 市场过热 → 见顶信号）  
**有效窗口**：未来 1~5 个交易日

### F11：成交量历史分位数因子 — Vol_Pct_Rank

**核心逻辑**：`Pct = Rank(Volume, 60d) / 60`

- **分位数高**：当前成交量处于60日高位 → 放量 → 关注度高 → 趋势可能延续
- **分位数低**：当前成交量处于60日低位 → 缩量 → 无人问津 → 可能反转

**为什么低BARRA**：BARRA Liquidity 因子直接用**换手率的平方根**——度量的是流动性的绝对水平。Vol_Pct_Rank 度量的是**成交量的相对位置**——当前量能在自身历史中的位置。一只大盘蓝筹可以绝对换手率很低（BARRA Liquidity 低）但处于其自身60日高位（Vol_Pct_Rank 高），两者信号完全不同。

**因子类型**：量价因子  
**IC方向**：正向（量在价先 — 放量 → 趋势确认）  
**预期IC均值**：0.01 ~ 0.03  
**ICIR**：0.2 ~ 0.4  
**有效窗口**：未来 1~10 个交易日

### F12：多周期均线离散度因子 — MA_Dispersion

**核心逻辑**：计算5条SMA（5/10/20/30/60）的**变异系数**：

```
MA_5   = SMA(Close, 5)
MA_10  = SMA(Close, 10)
MA_20  = SMA(Close, 20)
MA_30  = SMA(Close, 30)
MA_60  = SMA(Close, 60)

Dispersion = Std(MA_5, MA_10, MA_20, MA_30, MA_60) / Mean(...)
```

- **高离散度（发散）**：均线多头发散 → 趋势明确 → 趋势延续
- **低离散度（粘合）**：均线缠绕 → 变盘在即 → 等待方向选择

**为什么低BARRA**：BARRA Momentum 看价格收益率的方向（涨/跌）。MA_Dispersion 看价格在不同周期均线的**一致性**——也就是趋势的"强度"而非趋势的"方向"。一只已经跌了很久的股票可能均线高度发散向下（本因子看空），但 MACD 已经金叉（BARRA Momentum 看多）。这种发散度 vs 方向的维度分离是关键。

**因子类型**：趋势结构因子  
**IC方向**：正向（均线发散 → 趋势明确 → 趋势延续）  
**预期IC均值**：0.01 ~ 0.03  
**ICIR**：0.2 ~ 0.4  
**有效窗口**：未来 5~20 个交易日

## 四、使用建议

### 因子处理流程（使用前必做）

```
原始因子值
    ↓
MAD去极值（±5倍中位数绝对偏差）
    ↓
行业中性化（申万一级行业虚拟变量回归取残差）
    ↓
市值中性化（对 ln_cap 回归取残差）
    ↓
截面Z-Score标准化（均值为0，标准差为1）
    ↓
等权或ICIR加权合成
```

### 推荐组合方案

```
方案 A（稳健型）:
  30% 技术结构组 (F01 + F03 + F06)
  25% 微观结构组 (F04 + F05 + F11)
  20% 波动结构组 (F02 + F08 + F09)
  25% 高阶情绪组 (F07 + F10 + F12)

方案 B（进取型）:
  40% 技术结构组 
  20% 高阶情绪组
  20% 微观结构组
  20% 波动结构组
```

### 与主流因子结合的注意事项

```
✅ 可与 BARRA 因子组合使用（低相关意味着真正的组合多样化）
✅ 适用于 A 股全市场，剔除 ST/停牌股票
⚠️ 小盘股信号更强（操纵痕迹少，微观结构更丰富）
⚠️ 震荡市效果优于单边趋势市
❌ 不要在整个市场极端波动期间过度依赖（2024年2月流动性危机期间失效）
```
